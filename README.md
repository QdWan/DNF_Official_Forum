# DNF_Official_Forum
## 本项目仅用于个人学习
### DNF官方论坛模仿（JSP+Servlet+maven）
#### 工具+环境：
##### IDEA 
##### Tomcat 8.0
##### JDK1.8

#### 暂时完成的功能
* 用户的注册（昵称，手机号码的实时校验）
* 用户的登录
* 用户发帖
* 用户回帖
* 查看其他用户个人信息
* 查看其他用户发的主题帖
* 查看徽章仓库
* 查看自己的徽章
* 用户修改头像，个人信息
* 帖子分模块展示
* 服务器开启时，自动同步本地资源。关闭服务器备份文件。（防止上传的头像文件丢失）
* 统计在线人数和历史访问量
